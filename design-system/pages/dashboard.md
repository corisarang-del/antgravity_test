# Dashboard Page Overrides

## 적용 범위
- 이 문서는 `design-system/MASTER.md`를 기본으로 하고,
  대시보드 화면에서 필요한 규칙만 오버라이드한다.

## 목표
- 실시간성 인지 강화
- 데이터 밀도 유지 + 초보 사용자 이해도 확보
- 유료/고레벨 사용자 기능 차이를 명확히 표시

## 페이지별 적용/비적용 예시
- 적용: 실시간 배지, 캐시 상태 패널, 근거 지표 카드
- 비적용: 히어로형 대형 장식 배너, 과한 글로우 배경

## 정보 구조 오버라이드
1. 상단 고정 바
- 종목 검색, 사용자 등급, 마지막 갱신 시각 표시

2. 핵심 KPI 영역
- 예측 방향, 신뢰도, 변동성 3카드 우선 배치
- 신뢰도는 퍼센트 + 등급 배지 동시 표시

3. 근거 영역
- 기술지표/감성/외부지표를 탭 또는 아코디언으로 분리
- 각 근거 항목에 데이터 기준 시각 표시

4. 캐릭터 코치 영역
- 화면 우측(데스크톱) 또는 하단(모바일) 배치
- 본문 데이터보다 시각 우선순위가 높아지지 않게 크기 제한

## 상호작용 오버라이드
- 실시간 갱신 가능 여부를 사용자 등급 기반으로 노출
- 기본 등급: 지연 갱신 안내 배지 표시
- 유료/고레벨: 실시간 갱신 활성 배지 표시
- 캐싱 상태(적용 여부, TTL, 마지막 갱신) 패널 필수

## 상태 표현 오버라이드
- 로딩: 스켈레톤 + "데이터 동기화 중" 문구
- 빈 상태: 관심 종목 추가 CTA 제공
- 오류: 네트워크/권한/데이터지연 상태를 분리해 안내
- 충돌 상태: 설정 동시 수정 감지 시 재확인 모달 노출

## 차트 오버라이드
- 기본 기간 프리셋: 1D, 1W, 1M, 3M
- 이벤트 마커: 실적 발표일/주요 뉴스 시점만 최소 표기
- 색상: 상승 success, 하락 danger, 중립 muted-foreground 고정

## KPI 품질 기준
- 첫 화면 핵심 정보 인지: 3초 이내
- 종목 전환 후 주요 카드 갱신: 2초 이내
- 오류 상태에서 사용자가 다음 액션 이해: 1단계 이내

## 금지 사항
- 대시보드에서 과도한 장식 애니메이션 사용 금지
- CTA를 예측 결과보다 시각적으로 과하게 강조 금지
- 차트와 카드 간 의미 중복 문구 남발 금지
