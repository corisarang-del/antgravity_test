# Ant Gravity Design System Master

## 온보딩 요약
- 이 문서는 앤트그라비티 웹뷰 디자인의 공통 기준이다.
- 페이지별 예외는 `design-system/pages/*.md`에서만 정의한다.
- 구현 전 우선 확인 순서: 공통 규칙 -> 토큰 -> 페이지 오버라이드.

## 제품 성격
- 웹뷰 기반 AI 주가 예측 SaaS
- 신뢰감 있는 금융 UX + 친근한 개미 캐릭터
- 데이터 가독성 우선, 캐릭터는 보조 내비게이터 역할

## 톤 앤 매너
- 키워드: 신뢰, 친근, 깔끔, 절제된 유머
- 금지 카피: 수익 보장/확정 매수·매도 유도/공포 조장 문구
- CTA/본문 우선순위: 본문 데이터 해석 > CTA 강조
- 장식 비중 제한: 장식 요소는 화면 시각 면적의 20% 이내 유지

## 타이포그래피
- Heading/Body: Plus Jakarta Sans
- 본문 최소 16px, line-height 1.6
- 숫자 데이터는 semibold + tabular-nums 사용

## 레이아웃
- 모바일: 단일 컬럼 카드
- 데스크톱: 12컬럼 그리드
- 우선순위: 예측 방향/신뢰도 > 근거 지표 > 캐릭터 피드백

## 웹뷰 제약 표준
- safe-area inset을 기본 패딩으로 적용한다.
- 터치 타깃은 최소 44x44px를 유지한다.
- 모바일 375px에서 가로 스크롤이 발생하면 안 된다.
- 고정 요소는 콘텐츠 가림 방지 오프셋을 포함한다.

## 라이트/다크 공통 규칙
- 라이트/다크 모두 동일 정보 구조를 유지한다.
- 상태 색상 의미는 모드와 무관하게 고정한다(상승/하락/중립).
- 다크 모드에서 과도한 네온 효과로 텍스트 가독성을 저해하면 안 된다.

## 컴포넌트 규칙
- 버튼: 최소 44x44px, 로딩 중 중복 클릭 방지
- 카드: 데이터 카드와 캐릭터 카드 분리
- 차트: 수치/요약 문구 동시 노출, 색상만으로 의미 전달 금지

## 상태/피드백
- 로딩: 스켈레톤 + "데이터 동기화 중" 문구
- 네트워크 오류: 재시도 액션 제공
- 데이터 지연: 마지막 갱신 시각 표시
- 권한 오류: 업그레이드 또는 재인증 안내
- 빈 상태: 다음 행동 CTA 제공

## 접근성
- 본문 텍스트 대비 4.5:1 이상
- 포커스 링 항상 표시
- 아이콘 전용 버튼 aria-label 필수
- prefers-reduced-motion 대응

## 보안/신뢰
- GPS 등 민감한 개인정보 저장 금지
- 투자 조언 단정 문구 금지
- 예측 결과에는 근거 지표 + 리스크 문구를 반드시 함께 표시

## 금지 패턴
- 과도한 장식 애니메이션
- 감정 과잉 카피
- 색상만으로 상태 전달
- 데이터 근거 없는 CTA 과강조

## 캐릭터 가이드
- 역할: 분석 도우미
- 상태: 기본/경고/축하 3가지
- 원칙: 캐릭터 단독으로 매수/매도 유도 금지

## 품질 체크
- SVG 아이콘만 사용(이모티콘 금지)
- 모든 클릭 요소 cursor-pointer 적용
- 모바일 375px 가로 스크롤 없음
- 라이트/다크 모두 대비 기준 충족
