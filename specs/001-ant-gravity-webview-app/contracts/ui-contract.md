# UI Contract - Ant Gravity Webview

## 공개 화면(비로그인 허용)
- 대시보드(요약 카드 + 예측 차트)
- 예측 근거 탭(핵심 지표 카드)
- 예측 이력 탭(과거 정확도/성과 시각화)

## 로그인 필수 화면
- 관심 종목 관리
- 알림 설정
- 인앱 알림함(읽음/미읽음)

## UI 상태 계약
- 데이터 상태 배지: NORMAL | DELAYED | FAILED
- 빈 상태: 데이터 없음 안내 + 대체 행동 버튼
- 오류 상태: 사용자 친화 메시지 + 재시도 버튼

## 디자인 계약
- 톤: 신뢰(기본) + 친근/퍼니(보조)
- 정보 우선순위: 예측 결과 > 신뢰도 > 근거 지표 > 이력
- 웹뷰 우선 반응형: 작은 화면 탭 네비게이션 최적화
